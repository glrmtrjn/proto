HardNoc: Compilation guide for Dummies

for Xilinx University Program - Virtex-II Pro Development System - Curriculum-on-a-chip - Digilent

1 - Uncompress HardNoC.rar.
	1.1 - Download HardNoC last version at SVN.
	1.2 - uncompress HardNoC.rar at your preference folder (ex: <C:\myprojects>)
2 - Compilling HardNoC:
	2.1 - Open HardNoC/HardNoC/HardNoC.ise (ex: <C:\myprojects\HardNoC\HardNoC\HardNoC.ise>) - ISE'll open.
	2.2 - Select HardNoC VHDL at Sorces window;
	2.3 - click with mouse right button "<Configure Target Device>/<Manage Configuration Project (iMPACT)>" and select "Rerun all"
	2.4 - while compilling, power on Virtex-II Pro Development System and connect USB cable on computer
	2.5 - Select "Configure devices using Boundary-Scan (JTAG)" and "Automatically connect to a cable and identify Boundary-Scan chain" and click Finish
	2.6 - For xcf32p don't select any file (click Cancel)
	2.7 - For xccace don't select any file (click Cancel)
	2.8 - For xc2vp32 select hardnoc.bit and click OK;
	2.9 - Confirm the Object Files (only hardnoc.bit) and click OK
	2.10 - Click OK in the Device Programming Properties
	2.11 - Using mouse, right click over xc2vp30 and click "Program" - Program Succeeded will appear
3 - Compilling Serial Software
	3.1 - Open Serial_Software folder (ex: <C:\myproject\HardNoC\Serial_Software>)
	3.2 - Uncompress "commapi.rar"
	3.3 - Copy all three files to "lib" folder on Java Development Kit folder (ex: <C:\Program Files\Java\jdk1.6.0_25\lib>)
	3.4 - Copy all three files to "jre\lib" folder on Java Development Kit folder (ex: <C:\Program Files\Java\jdk1.6.0_25\jre\lib>)
	3.5 - Copy all three files to "bin" folder on Java Development Kit folder (ex: <C:\Program Files\Java\jdk1.6.0_25\bin>)
	3.5 - Edit compile.bat:
		3.5.1 - Change the "path" and the "classpath" to the Java Development Kt installed in your computer.
	3.6 - execute "compile.bat"
